### **Общее условие для всех вариантов:**

Добавьте в класс `DemoArray` переопределение:

-  Бинарного минуса (из всех элементов массива вычитается заданное число);

-  Операции `&` (поэлементно сравнивает два массива: если соответствующие элементы попарно совпадают, возвращает `true`, иначе `false`).

#### **Варианты для Задания 2:**

1. **Вариант 1:** Добавить метод `AreAllPositive()`, возвращающий `true`, если все элементы > 0 (аналог `true`, но как метод).

2. **Вариант 2:** Добавить метод `AreAllEven()`, возвращающий `true`, если все элементы чётные.

3. **Вариант 3:** Добавить метод `Min()`, возвращающий минимальный элемент массива (если массив пуст -- выбросить `InvalidOperationException`).

4. **Вариант 4:** Добавить метод `Max()`, возвращающий максимальный элемент массива.

5. **Вариант 5:** Добавить метод `Contains(int value)`, проверяющий, содержится ли заданное значение в массиве.

6. **Вариант 6:** Добавить метод `CountZeros()`, возвращающий количество нулевых элементов.

7. **Вариант 7:** Добавить свойство `IsNotEmpty`, возвращающее `true`, если длина массива > 0.

8. **Вариант 8:** Добавить метод `Copy()`, возвращающий новый `DemoArray` -- точную копию текущего (аналог конструктора копирования, но как метод).

9. **Вариант 9:** Добавить метод `Reverse()`, возвращающий новый `DemoArray` с элементами в обратном порядке.

10. **Вариант 10:** Добавить метод `GetPositiveCount()`, возвращающий количество положительных элементов.

11. **Вариант 11:** Добавить метод `GetNegativeCount()`, возвращающий количество отрицательных элементов.

12. **Вариант 12:** Добавить метод `IsEmpty()`, возвращающий `true`, если длина массива равна 0.
