### **Общее условие для всех вариантов:**

Разработать класс `DemoArray`, реализующий одномерный массив целых чисел (`int[]`), в котором содержатся следующие элементы:

-  конструктор, позволяющий создать объект-массив заданного размера (`new DemoArray(5)` -> массив из 5 нулей);

-  конструктор, позволяющий инициализировать объект-массив обычным массивом (`new DemoArray(new int[] {1, 2, 3})`);

-  свойство `Length`, возвращающее размерность массива;

-  индексатор `this[int i]`, позволяющий читать и записывать элемент по индексу (с проверкой границ);

-  метод `Print()`, выводящий содержимое массива в консоль (через пробел);

-  перегрузка унарной операции `-` (все элементы меняют знак: `[1, -2]` -> `[-1, 2]`);

-  перегрузка операции `++` (все элементы увеличиваются на 1);

-  перегрузка операций `true` и `false` (`true`, если все элементы > 0, иначе `false`).

### **Варианты для Задания 1:**

1. **Вариант 1:** Добавить перегрузку унарной операции `+` (прибавляет число ко всем элементам).

2. **Вариант 2:** Добавить перегрузку операции `--` (все элементы уменьшаются на 1).

3. **Вариант 3:** Добавить перегрузку операции `~` (инверсия побитовая: каждый элемент заменяется на `~x`).

4. **Вариант 4:** Добавить перегрузку операции `!` (возвращает `true`, если все элементы ≤ 0).

5. **Вариант 5:** Добавить явное преобразование `DemoArray` -> `int[]` (возвращает копию внутреннего массива).

6. **Вариант 6:** Добавить неявное преобразование `int[]` -> `DemoArray`.

7. **Вариант 7:** Добавить явное преобразование `DemoArray` -> `int` (возвращает сумму всех элементов).

8. **Вариант 8:** Добавить перегрузку операции `*` как унарной (внутренний массив умножается поэлементно на -1).

9. **Вариант 9:** Добавить свойство `Sum`, возвращающее сумму элементов.

10. **Вариант 10:** Добавить свойство `IsZero`, возвращающее `true`, если все элементы равны 0.

11. **Вариант 11:** Добавить метод `SetAll(int value)`, устанавливающий всем элементам заданное значение.

12. **Вариант 12:** Добавить конструктор копирования: `public DemoArray(DemoArray other)`.
