Вариант - 1\
Создать класс `Human` с полями: ФИО, возраст. Реализовать метод вывода информации в консоль (например: `"Иванов И.И., 20 лет"`).\
Создать производный класс `Student` с полем -- номер группы. Переопределить метод вывода, добавив номер группы: `"Иванов И.И., 20 лет, группа ПИ-201"`.

Вариант - 2\
Создать класс `Human` с полями: имя, дата рождения (в виде строки `"дд.мм.гггг"`). Реализовать метод `GetAge()`, возвращающий возраст в годах (без учёта месяцев/дней -- упрощённо: текущий год минус год рождения).\
Создать производный класс `Teacher` с полем -- предмет преподавания. Добавить метод `Teach()`, выводящий: `"Учитель {имя} преподаёт {предмет}"`.

Вариант - 6\

Создать производный класс `Traveler` с полем -- список посещённых городов (можно использовать `List<string>` или просто строку). Добавить методы:

-  `AddCity(string city)` -- добавляет город в список,

-  `HasVisited(string city)` -- возвращает `true`, если город был посещён.
