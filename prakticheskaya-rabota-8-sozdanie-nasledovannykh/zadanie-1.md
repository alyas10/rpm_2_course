Вариант - 1\
Создать класс `Pair` (пара целых чисел); определить метод умножения на число и операцию сложения пар: `(a, b) + (c, d) = (a + c, b + d)`.\
Определить класс-наследник `Money` с полями: рубли и копейки.\
Переопределить операцию сложения и определить методы вычитания и деления денежных сумм.

Вариант - 2\
Создать класс `Pair` (координаты точки на плоскости: x, y). Определить метод сдвига точки на заданный вектор (dx, dy).\
Определить класс-наследник `PolarPoint` с полями: угол (в градусах) и радиус. Добавить метод перевода в декартовы координаты и вывода результата.

Вариант - 3\

Определить класс-наследник `MixedFraction` с полями: целая часть, числитель, знаменатель. Добавить метод преобразования в неправильную дробь (возвращающий `Pair`).

Вариант - 4\
Создать класс `Pair` (ширина и высота прямоугольника). Определить метод вычисления площади.\
Определить класс-наследник `Square` с одним полем -- длина стороны. Переопределить метод площади и добавить метод вычисления периметра.

Вариант - 5\
Создать класс `Pair` (показания счётчика за прошлый и текущий месяц). Определить метод вычисления разности (расход за месяц).\
Определить класс-наследник `WaterMeter` с дополнительным полем -- тариф (руб./м³). Добавить метод расчёта стоимости потреблённой воды.

Вариант - 6\

Определить класс-наследник `Event` с полем -- название события. Добавить метод вывода информации в формате: `"Событие X: 2020–2023"`.

Вариант - 7\
Создать класс `Pair` (часы и минуты -- время суток). Определить метод добавления заданного числа минут с учётом переполнения (перехода в следующие часы/сутки).\
Определить класс-наследник `AlarmClock` с полем -- флаг включён/выключен (bool). Добавить метод `ToString()`, возвращающий строку вида `"10:25 (будильник включён)"`.

Вариант - 8\

Определить класс-наследник `StudentGrade` с полем -- ФИО студента. Добавить метод вывода в формате: `"Иванов: 85/90 → 87.5"`.

Вариант - 9\
Создать класс `Pair` (длина и ширина земельного участка в метрах). Определить метод вычисления площади в сотках (1 сотка = 100 м²).\
Определить класс-наследник `GardenPlot` с полем -- тип культуры (строка). Добавить метод `GetDescription()`, возвращающий строку вида `"Участок 20×30 м под картофель (6 соток)"`.

Вариант - 10\

Определить класс-наследник `ForceVector` с дополнительным полем -- масса тела. Добавить метод вычисления ускорения по формуле `a = |F| / m`.

Вариант - 11\
Создать класс `Pair` (год и номер месяца). Определить метод, возвращающий количество дней в указанном месяце (учитывать високосный год для февраля).\
Определить класс-наследник `CalendarMonth` с полем -- название месяца (строка). Добавить метод вывода в формате: `"Март 2024 (31 день)"`.

Вариант - 12\
